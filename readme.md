# X (Twitter) to Commons

Upload images from X (Twitter) to Wikimedia Commons.

## About

Odia Wikipedia Community collaborated with the government of Odisha and brought many of the government's social accounts under CC-BY-4.0 license. This helps the Odia Community to upload images from those government X (Twitter) handles to Wikimedia Commons directly and easily. This tool is useful for anyone who makes their X (Twitter) handle under CC license and has an OTRS token.

## Requirements

- PHP 8.4+
- Laravel 11.x
- Composer

## Installation

```bash
composer install
cp .env.example .env
php artisan key:generate
```
